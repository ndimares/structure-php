## SDK Installation

### Composer

To install the SDK first add the below to your `composer.json` file:

```json
{
    "repositories": [
        {
            "type": "github",
            "url": "https://github.com/structure-ac/structure-php.git"
        }
    ],
    "require": {
        "structure/Structure": "*"
    }
}
```

Then run the following command:

```bash
composer update
```
